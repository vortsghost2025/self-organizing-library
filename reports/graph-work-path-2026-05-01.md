## Methodology

1. Load `data/site-index.json` (entries + cross-references + tag index)
2. Run `computeAuthorityEdges()` — classifies cross-refs and tag-group connections into authority types
3. Run `computeNodeStatuses()` — computes UNVERIFIED/VERIFIED/CONFLICTED/QUARANTINED + counts
4. Run `computeGovernanceDepths()` — computes governanceLayer, authorityDepth, bridgeState
5. Classify each node into signal buckets based on edge composition, status, and governance metadata
6. Score priority: P0 = misleading/unsafe, P1 = status/governance gap, P2/P3 = taxonomy cleanup

### Artifact-Class Detection
