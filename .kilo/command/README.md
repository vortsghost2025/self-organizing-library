# Library Lane Commands

Structured command templates used to reduce variance and enforce deterministic workflows.

## Available Commands

- `phase-commit-intent.md` - pre-commit intent and verification contract
- `cross-lane-delivery-check.md` - canonical-path inbox delivery validation
- `enforcement-proof-gate.md` - runtime-path enforcement proof checklist
- `lane4-release-intake-check.md` - consume-only-pinned release intake from Kernel-Lane

## Usage Rule

If a task matches one of these workflows, use the template instead of ad-hoc command text.
